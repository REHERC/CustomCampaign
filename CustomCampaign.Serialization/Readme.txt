Using the JSON serializer

	PM> Install-package Newtonsoft.Json
	Add the "SERIALIZER_JSON" compilation symbol.


Using the XML serializer

	Add the "SERIALIZER_XML" compilation symbol.


Using the BIN serializer

	Add the "SERIALIZER_BIN" compilation symbol.


Using the YAML serializer

	PM> Install-package YamlDotNet
	Add the "SERIALIZER_YAML" compilation symbol.


Using the CSV serializer
	
	PM> Install-package FileHelpers
	Add the "SERIALIZER_CSV" compilation symbol.
